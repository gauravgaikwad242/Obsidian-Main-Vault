## How to access AWS 
1. UI - management console 
2. CLI - protected by access keys 
3. SDK - protected by access key - to call aws from application or api 

- Access key id = user name 
- secrete access key = password 

SDK = software development kit 
 - different for each application 
 - skd - java , c , c++ etc 
 - mobile sdk 
 - IoT device sdk
