1. Compiler approach for Auto-Boxing 

|Primitive Data type   |  --- valueOf() --->  |   Wrapper Class Object|
|---|---|---|
```java 
//eg
byte b = 10;
Byte b1 = Byte.valueOf(b);
Byte b2 = b;
```

2. Compiler approach for Auto-unboxing 

|Wrapper Class Object |  --- abcValue() --->  |  Primitive Data type|
|---|---|---|

eg for un-boxing
```java 
Byte b = new Byte("10");
byte b1 = b.byteValue();
```
